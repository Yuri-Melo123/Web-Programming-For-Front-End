# Web-Programming-For-Front-End

Você foi convidado para criar um Web site que consome dados de uma API referente ao mundo de desenhos em quadrinhos, para isso você deverá planejar e desenvolver uma página contendo conteúdos relevantes ao assunto escolhido, esse projeto trará também dados em tempo real do desenho escolhido que teve muito sucesso ao longo dos anos. 

Aplicativo criado a partir do repositório:
https://github.com/Yuri-Melo123/Trabalho-Design-Web.git


O Objetivo original era exibir apenas os personagens conforme o arquivo: ./js/version1/script.js, porém, foi acrescentado uma barra de busca: ./js/version1/personagens.js e por fim, uma seção para listar os episódios: ./js/version1/episódios.js.

Depois de pronto eu achei melhor criar um novo arquivo para armazenar as funções utilizadas tanto para os personegens quanto para os episódios no arquivo: ./js/version2/ui.js.
